# 📱 카드 문자 자동 전송 세팅 가이드 (안드로이드 / MacroDroid)

카드 결제 문자가 오면 → 자동으로 구글시트로 보내 → 장부관리 앱 "사업비 후보"에 뜨게 만드는 설정입니다.
한 번만 세팅하면 그 뒤로는 신경 쓸 필요 없습니다. **약 10분** 소요.

---

## 준비물 2가지

### 1. GAS 웹앱 주소 (URL)
- 장부관리 앱 → **⚙ 시트설정** 에 입력했던 그 주소입니다.
- `https://script.google.com/macros/s/AKfy............/exec` 형태.
- 이 주소를 미리 복사해 두세요. (아래 5번에서 붙여넣음)

### 2. MacroDroid 앱
- Play 스토어에서 **MacroDroid** 검색 → 설치.
- 무료 버전으로 충분합니다. (매크로 5개까지 무료 — 우리는 1개만 씀)

---

## 따라 하기

### 1단계 — 새 매크로 만들기
1. MacroDroid 실행 → 하단 **매크로** → 우측 아래 **+ (추가)** 버튼

### 2단계 — 트리거(언제?) 설정
"문자가 올 때마다 실행" 이라는 조건입니다.
1. **트리거 추가 (+)** → **연결** 또는 **수신** 카테고리 →
2. **SMS 수신** (영문: *Received SMS*) 선택
3. 발신자: **모든 연락처 / 모든 번호** 선택 (특정 카드사만 받고 싶으면 카드사 번호 입력 — 비워두면 전부)
4. 확인

### 3단계 — 액션(무엇을?) 설정
"받은 문자를 인터넷 주소로 보내라" 입니다.
1. **액션 추가 (+)** → **연결(Connectivity)** 카테고리 →
2. **HTTP 요청 (HTTP Request)** 선택
3. 아래처럼 입력:

   | 항목 | 값 |
   |---|---|
   | **요청 방식 (Method)** | `POST` |
   | **URL** | (위에서 복사한 GAS 주소 붙여넣기) |
   | **콘텐츠 타입 (Content Type)** | `text/plain` 또는 비워둠 |
   | **본문 (Body / Content)** | 아래 내용 복사 ↓ |

4. **본문(Body)** 칸에 이걸 그대로 붙여넣기:
   ```
   {"secret":"a4463116!","source":"sms","receivedAt":"[time]","body":"[sms_message]"}
   ```

   > 💡 `[sms_message]`, `[time]` 은 MacroDroid의 **변수(자석 아이콘 🧲)** 에서 골라 넣어야 합니다.
   > - 본문 칸에서 자석 아이콘 클릭 → **SMS 메시지** 선택 → `[sms_message]` 자동 삽입
   > - 마찬가지로 **시스템 → 현재 시각**(또는 날짜/시간) → `[time]` 삽입
   > - 직접 `[sms_message]` 라고 타이핑해도 됩니다.

### 4단계 — 저장
1. 우측 위 ✓ (저장) → 매크로 이름: `카드문자→장부` 같은 걸로 저장
2. 매크로 목록에서 **켜짐(ON)** 상태인지 확인

### 5단계 — 권한 허용
- 처음 실행 시 MacroDroid가 **SMS 읽기 권한**을 요청합니다 → **허용**.
- 배터리 최적화에서 MacroDroid를 **제외**해 두면 백그라운드에서 안 꺼집니다.
  (설정 → 배터리 → MacroDroid → 제한 없음)

## 테스트

1. 본인 폰으로 아무 문자나 받아보거나, 실제 카드 결제를 해봅니다.
2. 잠시 후 **구글시트**를 열어 봅니다:
   - `sms_raw` 시트 → 받은 문자 원문이 한 줄 쌓이면 ✅ 전송 성공
   - `expense_candidates` 시트 → 롯데카드 결제면 한 줄 추가됨
3. 장부관리 앱 → **📱 사업비 후보** → **↻ GAS 동기화** 버튼 → 후보 목록에 뜸!

## 자주 묻는 것

**Q. 모든 문자가 다 전송되나요? 개인 문자도?**
- MacroDroid는 모든 문자를 GAS로 보내지만, **GAS가 걸러냅니다.**
  - 롯데카드(사업용) → 후보로 등록
  - 현대카드(개인용) → "제외됨"으로 기록만
  - 그 외 일반 문자 → 무시(skip), 후보에 안 뜸
- 단, 모든 문자 원문은 `sms_raw` 시트에 남습니다. 신경 쓰이면 트리거에서 카드사 번호만 지정하세요.

**Q. 카드가 더 있어요 (사업용/개인용 추가)**
- 구글시트 Apps Script의 `Code.gs` 맨 위:
  ```js
  const BUSINESS_CARDS = ['롯데카드'];   // 사업용 추가: ['롯데카드','신한카드']
  const PERSONAL_CARDS = ['현대카드'];   // 개인용 추가
  ```
  카드 이름을 넣고 다시 배포하면 됩니다.

**Q. 자동이 귀찮거나 안 되면?**
- 앱 "사업비 후보 → **+ 직접 추가**"에 카드 문자를 복사·붙여넣으면 날짜·금액·가맹점이 자동으로 채워집니다. MacroDroid 없이도 사용 가능.

## 보안 메모
- 본문의 `"secret":"a4463116!"` 는 아무나 가짜 데이터를 못 넣게 막는 암호입니다.
- 이 값은 `Code.gs` 의 `SMS_SECRET` 과 똑같아야 합니다. (현재 둘 다 `a4463116!`)
- 더 안전하게 하려면 `Code.gs`의 `SMS_SECRET` 을 다른 값으로 바꾸고, MacroDroid 본문의 secret도 같이 바꾸세요.
